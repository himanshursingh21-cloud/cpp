My C++ Learning Journey 🚀
This repository is a collection of my C++ projects as I learn the language. These programs range from basic calculators to interactive games.

🛠 Projects Included:
Banking Program: A simulation of basic banking operations.

Quiz Game: An interactive trivia quiz.

Stone Paper Scissors: A classic game against the computer.

Calculator: A simple arithmetic calculator.

Hypotenuse Calculator: Computes the hypotenuse of a right-angled triangle.

Random Number Generator: Generates random numbers within a specified range.

Random Guessing Game: A game where the user guesses the computer's secret number.

Temperature Converter: Converts temperatures between Celsius and Fahrenheit.


💻How to Run:
1-Make sure you have a C++ compiler (like g++) installed.

2-Clone this repository to your local machine.

3-Open your terminal in the project folder.

4-Compile the desired file:
g++ filename.cpp -o output

5-Run the program:
./output (or output.exe on Windows)
